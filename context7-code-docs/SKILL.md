# Context7 Code Docs

Use this skill before implementing or explaining version-sensitive third-party APIs.

## Workflow

1. Identify the relevant library, framework, SDK, or integration. Determine the version from local files when possible.
2. Resolve the library with Context7.
3. Fetch focused docs or examples for the specific API, config, middleware, migration, or error.
4. Use only the necessary details in code or explanation.
5. Mention Context7 usage in chat history logging when that logging is available.

## Fallback

If Context7 is unavailable, proceed with local code and built-in knowledge only when reasonable. Mark version-sensitive details as uncertain, and prefer primary or official sources if another documentation path is available.
